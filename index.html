Smart-M3 distribution repo

For installing Smart-M3 do next steps:

  • Add next line into /etc/apt/sources.list file
    deb http://download.geo2tag.org/smart-m3-repo/ smart-m3 stable
    deb http://download.geo2tag.org/smart-m3-repo/ smart-m3 testing
  • Run wget -q -O - http://download.geo2tag.org/public.key | sudo apt-key add -
  • Run sudo apt-get update
  • Run sudo apt-get install redsibd sib-tcp libwhiteboard whiteboard whiteboard-sib-access to install SIB and both SIB and KP side transport modules

For initial testing Smart-M3 do next steps:

Installing Web-SIB-Explorer:

  • Dowload Web-SIB-Explorer
  • Unzip downloaded archive to any folder you want
  • Install python package manager by typing sudo apt-get install python-pip in terminal
  • Install Web-SIB-Explorer dependencies by sudo pip install flask flask-sqlalchemy

Testing:

  • Open 3 terminal windows and run commands: redsibd, sib-tcp, whiteboardd. Each command in separate window. If you want smart-m3 to be reachable from KPs(e.g. Web-SIB-Explorer) that are not running on the same device, run sib-tcp command with parameter --ip 0.0.0.0. Also you could specify PORT with --port PORT. So you can run smart-m3 and Web-SIB-Explorer on different machines.
  • In another terminal window go to the directory, where you unziped Web-SIB-Explorer
  • To run Web-SIB-Explorer type python explorer.py. Then open your web browser and go to http://127.0.0.1:5000. You can also pass the argument --ip and --port for running the application on a different IP and PORT
  • To connect to the SIB you can use either the IP or an IP:PORT combination. The default port for connecting to the SIB is 10010, and it will be used if no port is supplied. If you are only running sib-tcp locally, without any arguments, then you are running the SIB on 127.0.0.1:10010, and you can connect simply by typing 127.0.0.1.
