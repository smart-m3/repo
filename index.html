Smart-M3 distribution repo

For installing Smart-M3 do next steps:

  • Add next line into /etc/apt/sources.list file
    deb http://download.geo2tag.org/smart-m3-repo/ smart-m3 stable
    deb http://download.geo2tag.org/smart-m3-repo/ smart-m3 testing
  • Run wget -q -O - http://download.geo2tag.org/public.key | sudo apt-key add -
  • Run sudo apt-get update
  • Use sudo apt-get install for installing required components: redsibd, sib-tcp, libwhiteboard, whiteboard, whiteboard-sib-access

For initial testing Smart-M3 do next steps:

  • Go to http://eslab.github.io/Web-SIB-Explorer/ and download Web-SIB-Explorer
  • Install dependencies sudo pip install flask flask-sqlalchemy
  • To run it type python explorer.py. The open your web browser and go to _http://127.0.0.1:5000_. You can also pass the argument --ip and --port for running the application on a different IP and PORT

Testing:

  • To connect to the SIB you can use either the IP or an IP:PORT combination. The default port for connecting to the SIB is 10010, and it will be used if no port is supplied. If you are only running sib-tcp locally, without any arguments, then you are running the SIB on 127.0.0.1:10010, and you can connect simply by typing 127.0.0.1.
