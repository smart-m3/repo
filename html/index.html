Smart-M3 distribution repo
  • Install Smart-m3
  • Install Web-SIB-Explorer
  • Testing
  • Add next line into /etc/apt/sources.list file
    deb http://download.geo2tag.org/smart-m3-repo/ smart-m3 stable
    deb http://download.geo2tag.org/smart-m3-repo/ smart-m3 testing
  • Run wget -q -O - http://download.geo2tag.org/public.key | sudo apt-key add -
  • Run sudo apt-get update
  • Run sudo apt-get install redsibd sib-tcp libwhiteboard whiteboard whiteboard-sib-access to install SIB and both SIB and KP side transport modules
