Smart-M3 distribution repo
  • Install Smart-m3
  • Install Web-SIB-Explorer
  • Testing
  • Open 3 terminal windows and run commands in separate window: redsibd, sib-tcp, whiteboardd.
    If you want smart-m3 to be reachable from KPs(e.g. Web-SIB-Explorer) that are not running on the same device, run sib-tcp command with parameter --ip 0.0.0.0.
    Also you could specify PORT with --port PORT. So you can run smart-m3 and Web-SIB-Explorer on different machines.
  • In another terminal window go to the directory, where you unziped Web-SIB-Explorer.
  • To run Web-SIB-Explorer type python explorer.py.
  • Then open your web browser and go to http://127.0.0.1:5000. You can also pass the argument --ip and --port for running the application on a different IP and PORT.
  • To connect to the SIB you can use either the IP or an IP:PORT combination.
    The default port for connecting to the SIB is 10010, and it will be used if no port is supplied.
    If you are only running sib-tcp locally, without any arguments, then you are running the SIB on 127.0.0.1:10010, and you can connect simply by typing 127.0.0.1.
  • Click on Maintaince tab and then load Test Ontology.
  • Now on Benchmark tab run the benchmark.
